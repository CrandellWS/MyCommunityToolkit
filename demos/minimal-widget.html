Minimal Widget Integration

The simplest way to add a widget to your page

Quick Start Steps

  1. 1

    Include the CSS files

    Add the stylesheet links in your HTML <head>

  2. 2

    Add a container element

    Create a <div> with an ID where the widget will render

  3. 3

    Include the JavaScript files

    Add the script tags before your widget initialization code

  4. 4

    Initialize the widget

    Call Widgets.create() with your container and options

Complete Example

<!-- HTML -->
<!DOCTYPE html>
<html lang="en" data-theme="dark">
<head>
  <!-- Step 1: Include CSS -->
  <link rel="stylesheet" href="src/styles/design-system.css">
  <link rel="stylesheet" href="src/styles/animations.css">
  <link rel="stylesheet" href="src/styles/components.css">
</head>
<body>

  <!-- Step 2: Add container -->
  <div id="my-widget"></div>

  <!-- Step 3: Include JavaScript -->
  <script src="src/utils/api-client.js"></script>
  <script src="src/components/widgets.js"></script>

  <script>
    // Step 4: Initialize widget
    Widgets.create('big-wins', '#my-widget', {
      title: 'Recent Big Wins',
      limit: 5,
      refreshInterval: 30000
    });
  </script>

</body>
</html>

Available Widget Types

// Stat Card - Display a single statistic
Widgets.create('stat', '#container', {
  title: 'Active Users',
  icon: '👁',
  color: 'primary',
  fetchFn: async () => { return 1234; }
});

// Big Wins - Live feed of big wins
Widgets.create('big-wins', '#container', {
  limit: 5,
  roomId: 'optional-room-id'
});

// Momentum Meter - Room activity indicator
Widgets.create('momentum', '#container', {
  roomId: 'your-room-id',
  showLabels: true
});

// Missions List - Active missions
Widgets.create('missions', '#container', {
  limit: 5,
  showProgress: true
});

// Rate Limit Monitor - API usage
Widgets.create('rate-limit', '#container', {});

Additional Widgets

// Leaderboard Widget (requires leaderboard.js)
Leaderboard.create('#container', {
  title: 'Top Players',
  limit: 10,
  showPodium: true,
  showAvatars: true,
  missionId: 'optional-mission-id'

// Game Carousel Widget (requires game-carousel.js)
GameCarousel.create('#container', {
  title: 'Popular Games',
  limit: 20,
  showProvider: true,
  autoScroll: false
});

// User Profile Widget (requires user-profile.js)
UserProfile.create('#container', {
  username: 'StreamerName',
  showStats: true,
  showBadges: true
});

Tips

  • ✓

    Use a Room ID for your content

    Pass roomId: 'your-room-id' to filter data to your specific room

  • ✓

    Adjust refresh intervals

    Set refreshInterval in milliseconds (30000 = 30 seconds)

  • ✓

    Use themes

    Set data-theme="dark" on your HTML element for dark mode
